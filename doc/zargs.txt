INTRODUCTION                                    *zargs*

Zargs provides a few helpers for populating the |arglist|.


                                                *zargs-:Zargs*
:Zargs                   Open |netrw| to the zargs file list directory
:Zargs '%'               Populate the |arglist| from the current buffer
:Zargs [filename]        Populate the |arglist| from a file in the vim-zargs/zargs directory

                                                *zargs-zp*
zp                       Print out full path to current buffer
